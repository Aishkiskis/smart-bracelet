# Smart Voice Bracelet - Умный голосовой браслет

Браслет распознает команду "стоп" через микрофон и вибрирует(виртуально).

## Как запустить

1. Установить Python
2. Запустить: `python app.py`
3. Сказать "стоп"

## Файлы
- `app.py` - главная программа
- `train_stop.py` - обучение нейросети
- `stop_kws.py` - распознавание голоса
